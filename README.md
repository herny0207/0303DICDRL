# 林秉翰 | 個人網頁 (Personal Webpage)

這是一個為 **林秉翰** 設計的現代化單頁個人網頁，靈感來自 [Adelina](https://builder.hostinger.com/templates?preview=adelina&hideRating=1) 範本風格。

## ✨ 特色 (Features)

*   **極簡高對比設計**：採用深黑色背景與純白、丁香紫（Lilac）的配色方案，營造高端、專業且具藝術感的氛圍。
*   **即時時間顯示**：頁面頂端整合即時數位時鐘，隨時顯示目前時間。
*   **動態視覺效果**：
    *   **懸浮星型裝飾**：具備視差（Parallax）與隨機漂浮動畫的裝飾元素。
    *   **環境光晕**：背景微光動畫，增加頁面的層次感與現代感。
    *   **互動標籤**：具有滑鼠懸停效果的膠囊形狀標籤。
*   **全方位響應式設計 (RWD)**：
    *   **流體排版**：使用 `clamp()` 函數讓標題 (**林秉翰**) 隨螢幕大小流暢縮放。
    *   **彈性佈局**：在行動裝置上自動切換為垂直堆疊佈局，確保最佳閱讀體驗。
    *   **適配各種螢幕**：從 320px 的小手機到 4K 螢幕皆能完美呈現。

## 🛠️ 技術棧 (Tech Stack)

*   **HTML5**：語義化結構，利於 SEO。
*   **CSS3**：使用原生 CSS 變數（CSS Variables）與 Keyframe 動畫。
*   **JavaScript (Vanilla)**：處理即時時鐘邏輯與滑鼠追蹤視差效果。
*   **Google Fonts**：使用 'Outfit'、'Caveat' 與 'Noto Sans TC'。

## 📂 檔案結構 (File Structure)

*   `index.html`：網頁主要結構與內容。
*   `style.css`：定義視覺風格、排版與動畫。
*   `script.js`：實現時鐘顯示與互動效果。
